## Project Overview

**ChainLink Portfolio** - Cross-Chain Asset & Activity Tracker for Polkadot

### What It Does
- Connects to Polkadot.js wallet
- Queries 4 parachains (Polkadot, Astar, Moonbeam, Acala)
- Aggregates balances and activity
- Displays unified dashboard with charts
- Shows portfolio value in USD
- Mobile-responsive, modern UI
